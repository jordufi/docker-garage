The Roadmap

Docker Garage is growing. These AI-powered modules are coming — turning your home server into a full local AI stack.

🎨
Image Generation
Run Stable Diffusion or FLUX locally. Generate images on your own GPU, with no API costs and no limits.
🧊
Image → 3D
Convert 2D images into 3D models using local inference. Export to glTF or OBJ for use in any pipeline.
🧠
LLM + Open WebUI
One-click Ollama deployment with Open WebUI as the frontend. Run Llama, Mistral, Gemma, and more — locally.
🌐
Local Web Pages
Deploy and serve your own static or dynamic web apps within the Garage ecosystem, all behind a unified interface.
03

Up in Minutes

01
Clone the Repo
Pull the project from GitHub onto your home server or any machine running Docker.
02
Configure
Edit the .env file to match your setup — ports, volumes, GPU access, and service names.
03
Compose Up
Run docker compose up -d and the entire stack launches automatically.
04
Open the Garage
Navigate to localhost:3000 and start managing your services from a clean, unified dashboard.
